它是什么

一块尺寸精确、
排版够好看的画布

每个 <section class="card"> 就是一页。HTML/CSS 完全自由，想怎么设计就怎么设计。

  • 固定比例画布（小红书 / 方形 / PPT / 竖屏）
  • 原生像素渲染，导出 2x 高清 PNG
  • 纯静态，clone 即用，可挂 GitHub Pages
